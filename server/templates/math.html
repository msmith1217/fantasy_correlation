About The Math

You can read about how any of the stats are calculated on this website here:

Correlation

This stat is Pearson's Correlation Coefficient and it is calculated with a Python Library called NumPy. The main thing to know is that only games played by both the searched player and player in that row are included when this stat is calculated. If a player on the same team has played in the same game as the searched player they will be included in the results, so beware of small sample sizes skewing the correlations

Average Fantasy Points

This one as you can imagine is pretty simple, this is just an average of all games the player played in that season.
